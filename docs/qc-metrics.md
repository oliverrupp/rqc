# QC Metrics

## Library Complexity

### Mapping Rate
Fraction of reads assigned to annotated genes.

### Detected Genes
Number of detected genes.

### Normalized AUC
Transcript discovery efficiency independent of sequencing depth.

### Tail Gain
Additional genes discovered at high sequencing depth.

### Top-100 Transcript Fraction
Fraction of reads assigned to the 100 most abundant transcripts.

### Shannon Entropy
Expression diversity across genes.

### Effective Gene Count
exp(Shannon entropy).
